Canvas Chatbot

A chatbot built to assist students in navigating educational platforms like Canvas, Google Classroom, or Blackboard. It streamlines access to information such as assignments, grades, and teacher announcements, eliminating the complexity of these interfaces.

Motivation

Educational platforms often overwhelm new users with multiple options and links, making it difficult to stay organized. The chatbot simplifies this by providing direct access to grades, assignments, and messages, helping users save time and effort.

Prototyping Phase

Two prototypes were created: an AI-driven chatbot and a keyword-based chatbot. After analyzing user feedback during the first study, the keyword-based chatbot was chosen due to its simplicity and positive reception.

The MazeRunner

A fully functional game where players navigate a randomly generated maze. It challenges users to solve the maze and reach a predefined endpoint.

Development Process

The game was developed using Visual Studio Code and the Java Swing library. The maze generation leverages a recursive backtracking algorithm for dynamic and challenging layouts.

Gameplay

Players start at a designated point and use arrow keys to navigate the maze. Successfully reaching the endpoint triggers a victory message with a celebratory panel.
